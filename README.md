# MaJoT: Matrix Joint triangulation
Построение нейросетевого инструмента для совместной триангуляции двух матриц
Краткое содержание проекта. В многомерном анализе данных часто требуется найти такое
преобразование, которое одновременно приводит две матрицы к «удобному» для
дальнейшей работы и преобразований виду (например, к треугольному или диагональному).
Классические методы работают только для определённых классов матриц. Нейросетевой
подход позволяет искать нелинейные преобразования, которые одновременно упрощают
структуру обеих матриц, минимизируя специально сконструированный функционал
потерь. Пусть даны две квадратные матрицы $A$ и $B$, требуется построить обратимую матрицу
преобразования $T$, такую что матрицы $T^TAT$ и $T^TBT$ становятся одновременно
верхнетреугольными (или максимально близкими к треугольным). Сравнить результаты с
классическими методами.
